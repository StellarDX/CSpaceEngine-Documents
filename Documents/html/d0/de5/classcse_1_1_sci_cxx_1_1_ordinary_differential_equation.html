详细描述

常微分方程(ODE)求解器，此功能译自Scipy 以下是来自SciPy的原始功能描述：

本类对给定初始值的常微分方程组进行数值积分：

\[ \begin{cases} \mathbf{y}' = f(x, \mathbf{y}) \\ \mathbf{y}(x_0) = \mathbf{y}_0 \end{cases} \]

其中：

  • \(x\) : 一维自变量
  • \(\mathbf{y}(x)\) : \(N\)维向量值函数
  • \(f(x, \mathbf{y})\) : 待求解的微分方程

目标是在已知初始条件 \(\mathbf{y}(x_0)=\mathbf{y}_0\)时，近似计算满足微分方程组的 \(\mathbf{y}(x)\)

示例
洛特卡-沃尔泰拉方程的定义如下：

\[ \begin{cases} x'(t) = \alpha x - \beta x y \\ y'(t) = -\gamma y + \delta x y \end{cases} \]

以α = 1.5, β = 1, γ = 3, δ = 1为例，可以使用以下方法生成此函数：
float64 a = 1.5, b = 1, c = 3, d = 1;
auto lotkavolterra = [a, b, c, d](float64 t, std::vector<float64> z)
{
float64 x = z[0], y = z[1];
return std::vector<float64>{a*x - b*x*y, -c*y + d*x*y};
};
auto ODE = CreateODEFunction<...>(lotkavolterra, {10, 5}, 0, 15);
cout << '[';
for (int i = 0; i <= 300; ++i)
{
cout << (*ODE)((15. / 300.) * i) << ", ";
}
cout << "]\n";
